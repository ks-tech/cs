	差异化规范
	
    
		
    用一种统一的容易理解的前缀来代替以前常用的css hack，以提高代码可读性的一种规范更多。
    
		
  • 常用前缀: 
		
      
			
    • -moz-
      webkit 系列(谷歌:chrome/苹果:safari等)
      •  
			
    • -webkit-
      webkit 系列(谷歌:chrome/苹果:safari等)
      •  
			
    • -ms-
      ie9,ie10+ 等支持html5,css3 的IE浏览器
      •  
			
    • -o-
      Opera
      •  
			
    • -ie6- -ie7- -ie8- 
      不支持HTML5 的IE浏览器
      • 
			
    • -ie9-
      IE9(部分支持HTML5), -ie9- 优先级高于 -ms-
      • 
			
    • -ie-
       IE 系列浏览器, 优先级最低(会被-ms-, -ie?- 覆盖)
      •

    变形/动画/布局类功能增强
    
		
      
			
    • 溢出控制1D
      
				(max-width/max-height/min-width/min-height)
				
      控制超出大小的切除和不足大小的补充。
      
			
      • 
			
    • 固定定位0.5D
      
				(position:fixed)
				
      无风险，很快会支持，可能有附加规范
      
			
      • 
			
    • 变形增强2D
      
				(transform)
				
      完成2D变换的全部变换函数,对动画的支持还需优化
      
			
      • 
			
    • transition动画增强5D
      
				(transition-property)
				
      需要明确指定动画的属性，不支持 transition-property:all
      
			
      • 
			
    • animation动画增强5D
      
				(animation)
				
      貌似animation动画用javascript描述更合理，目前尚无支持计划。
      
			
      • 
			
    • 组合宽高增强0.5D
      
				(width:calc(50%-2px))
				
      貌似标准上载争论中,看需求确定何时实现
      
			
      • 
		
    
	
    • 
	
  • 
		
    背景图形类增强
    
		
      
			
    • 半透明支持
      (opacity)
				
      用户可以直接书写CSS3 opacity属性,系统自动调用filter实现(本系统有统一的fliter管理器,防止不同filter需求的冲突)。
      
			
      • 
			
    • PNG24 Alpha 背景图1D
      (background-image:xx.png)
				
      读取png源码,判断是否为png24 且包含alpha半透明,如存在,在ie6下会自动转化为filter实现兼容(本实现无相对路径问题)。
      
			
      • 
			
    • 透明背景色）0.5D
      
				(background-color:rgba(0,255,0,0.3);)
				
				
      如果背景色包含alpha通道，将通过filter渐变背景实现
      
			
      • 
			
    • 线性渐变4D
      (linear-gradient)
				
      目前采用filter实现，只能两个颜色，水平和垂直两个方向(不如CSS3强大)，后续可能尝试生成图片文件的方式
      
			
      • 
			
    • 阴影增强2D
      (box-shadow)
				
      计划用filter实现，不支持内阴影
      
			
      • 
			
    • 圆角增强4D
      (border-radius)
				
      目前采用filter＋vml实现，性能不佳，后续可能尝试动态生成图片文件的方式
      
			
      • 
			
    • 图形边框5D
      (border-image)
				
      有一定难度
